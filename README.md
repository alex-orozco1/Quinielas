# QRACKS ⚽

Create and manage sports prediction pools in minutes.

QRACKS is a lightweight web platform that lets anyone create their own sports pool, invite friends, collect predictions, automatically score every round, and keep historical standings across entire tournaments.

Designed for simplicity, privacy and trust.

---

## Features

✅ Create unlimited pools

✅ League-specific team selection

✅ Automatic scoring

✅ Live leaderboard

✅ Historical rankings

✅ Countdown until picks lock

✅ Secure participant PINs

✅ Admin dashboard

✅ Platform dashboard

✅ Automatic match results (TheSportsDB)

## How it works

1. Create a pool

2. Share your invite link

3. Participants join

4. Everyone submits predictions

5. Picks lock automatically

6. Admin publishes results

7. Standings update instantly
