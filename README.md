# ⚽ QRACKS

> Create and manage football prediction pools (quinielas) in minutes.

QRACKS is a modern web application for creating and managing private football prediction pools with friends, family, coworkers or communities.

The goal is simple:

**Make organizing a quiniela effortless.**

---

# Why QRACKS?

Most prediction pool tools are either:

- difficult to configure
- full of ads
- visually outdated
- require spreadsheets
- or are built around betting instead of friendly competition.

QRACKS focuses on simplicity.

Create a quiniela.

Invite your friends.

Make predictions.

See who wins.

---

# Core Features

## Administrator

- Create unlimited prediction pools
- Multiple matchdays (Jornadas)
- Manage participants
- Automatic standings
- Match result management
- Additional scoring rules
- Shareable links
- WhatsApp invitations
- Backups
- Platform administration

## Participants

- Simple join flow
- Match predictions
- Automatic score calculation
- Standings
- Match history
- Mobile-first experience

---

# Product Principles

QRACKS is built around a few principles.

## Simplicity

Every screen should answer:

- Where am I?
- What should I do?
- Why?

## Fast setup

An organizer should have a playable quiniela in under **5 minutes**.

## No learning curve

Participants shouldn't need instructions.

## Mobile First

Most users interact from WhatsApp.

QRACKS is designed accordingly.

# Development Philosophy

We intentionally avoid building features just because they are possible.

Every sprint must improve one measurable outcome.

Examples:

- Create a quiniela in under 5 minutes.
- Complete predictions in under 2 minutes.
- Understand the product in under 30 seconds.

---

# Security

QRACKS follows a server-side authorization model.

Sensitive operations are validated on the backend.

Credentials are never exposed through public APIs.

Security improvements are continuously prioritized before adding new functionality.

---

# Status

🚧 Active development.

Current focus:

**Security Minimum Viable Sprint**
